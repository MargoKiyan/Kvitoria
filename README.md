# Курсова робота 2026
## Тема: <i>"Розробка вебзастосунку "Квіторія" для колекціонерів кімнатних рослин".</i><br><br>Виконала: <i>ст. гр. ПД-21 Киян Маргарита</i>

"Квіторія" - це ASP.NET Core MVC WebApp на C# для колекціонерів рослин. Дані зберігаються у PostgreSQL через Entity Framework Core та Npgsql.

## Запуск

```bash
docker compose up -d postgres
```
```bash
DOTNET_CLI_HOME=".dotnet_home" dotnet run --urls http://localhost:5127
```
```text
http://localhost:5127
```

## Функції

- Авторизація і реєстрація через ASP.NET Core Identity.
- Розмежування доступу за ролями `User` і `Admin`.
- Особистий кабінет із незмінною поштою, датою народження і зміною логіну/пароля не частіше одного разу на місяць.
- Валідація логіну, email, пароля, дати народження та унікальності логіну/email.
- Обмеження поливу: одну рослину можна занотувати як политу лише один раз на дату.
- Автоматичний розрахунок наступного поливу за останнім поливом і частотою.
- Адмін-панель із керуванням користувачами, каталогом видів/форм/сортів рослин, зверненнями і звітами.
- Генерація звітів у `.txt` через FileIO та `.docx` через OpenXML.

Паролі нових акаунтів і seed-акаунтів зберігаються у форматі `SHA256$...`. Старі записи, створені до цієї зміни, автоматично перехешовуються після успішної зміни або перевірки пароля.

## Архітектура

- `Models` - OOP-домен: користувачі, рослини, каталог видів/форм/сортів, журнали догляду, зворотний зв'язок, enum-и та struct-и.
- `Data` - `KvitoriaDbContext`, PostgreSQL-конфігурація, Identity-таблиці та ініціалізація БД.
- `Services` - бізнес-операції, аналітика адміна, генерація звітів.
- `Controllers` - MVC-контролери для авторизації, кабінету, рослин, догляду, адмінки та звернень.
- `Views` - Razor-сторінки для користувача й адміністратора.
